🚀 Quick Start Guide
1. Use the left sidebar to navigate to each section (Hero, About, Skills, etc.)
2. Edit any field — changes are automatically tracked
3. Click Save & Publish in the top bar to apply all changes
4. Click Preview Portfolio at the bottom of the sidebar to see the live result
5. Both files (admin.html + index.html) must be in the same folder
